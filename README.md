# Steghide Automation 

# Description
```
A simple Bash Script which automates the extraction of Files from a JPG-File with Steghide !!
```

## Usage of StegHide Automator
```
1. bash steghide.sh
2. select JPG-FILE
3. look into your directory to see if any data has been extracted 

```

## Requirements
```
Steghide
```
